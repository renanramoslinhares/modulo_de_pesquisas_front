# Módulo de pesquisa [Parte 1 - Front]
Requisitos: NPM e Node (Testado com v12 e v15)

### Clone o repositório
```
git clone https://github.com/renanramoslinhares/modulo_de_pesquisas_front.git
```
### Em seguida rode o seguinte commando dentro do diretório `/modulo_de_pesquisas_front`
```
npm install && npm run serve
```

### Parte 2
Por padrão o app será iniciado na porta :8080 do seu localhost;
Só dará erro pois ele procurará a API na porta :8000 e ela ainda não não foi iniciada.
Parte 2: https://github.com/renanramoslinhares/modulo_de_pesquisas_back
